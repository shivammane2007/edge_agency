## 🚀 Overview

**Edge Agency** is a premium, high-fidelity landing page designed for a boutique AI deployment agency. Built with a focus on immersive aesthetics, cinematic animations, and high performance, it showcases a suite of cutting-edge services from custom AI integrations to full-scale digital rebranding.

## ✨ Key Features

- **🎬 Cinematic Experience**: Immersive background video (Galaxy MP4) and high-end parallax effects.
- **⚡ Performance First**: Blazing fast load times powered by Vite and React 19.
- **🎭 Motion Design**: Fluid animations and micro-interactions orchestrated via `framer-motion`.
- **💎 Premium UI**: A sleek, dark-themed interface utilizing custom glassmorphism and modern typography.
- **📱 Fully Responsive**: Seamless experience across all devices, from ultra-wide monitors to mobile screens.
- **📊 Dynamic Sections**: 
  - **Work Marquee**: Infinite scrolling showcasing agency partners.
  - **Case Studies**: Deep-dive into high-impact AI transformations.
  - **Pricing & Plans**: Tiered solutions for businesses of all scales.

## 🛠️ Tech Stack

- **Core**: [React 19](https://react.dev/), [Vite 6](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Configuration**: PostCSS, Autoprefixer

## 📦 Installation & Setup

Ensure you have [Node.js](https://nodejs.org/) installed on your machine.

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/shivammane2007/edge_agency.git
   cd edge_agency
   ```

2. **Install Dependencies**:
   ```bash
   npm install
   ```

3. **Configure Environment Variables**:
   Create a `.env.local` file and add your configuration (e.g., Gemini API keys if applicable):
   ```bash
   # .env.local
   GEMINI_API_KEY=your_api_key_here
   ```

4. **Run Development Server**:
   ```bash
   npm run dev
   ```

5. **Build for Production**:
   ```bash
   npm run build
   ```
